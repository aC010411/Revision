Wireless Data Transmission

Satellite communication

It can cover and build a network connetion in remote areas where wired connection is not accessible.

Worldwide Interoperability for Microwave access (WiMax)

  • A wireless technology that uses microwave to build network connection
  • Each microwave terminal must have a line-of-sight to the access point
  • Easily affected by weather

Wireless fidelity (Wi-Fi)

A LAN that offers Wi-Fi access are called Wireless LAN (WLAN).Additionally, an access point is a station that transmits and receives data in a wireless network. A Dual Band Router is a router that both supports 2.4GHz and 5GHz sognals simultaneously

Bluetooth

It has a slower transmission speed and smaller coverage than Wi-Fi.

NFC vs Bluetooth

  • NFC has lower power consumption than bluetooth
  • NFC has faster connection speed
  • NFC is more secure as it requires the card holder to put the payment device close to the reader

Encryption

Recent years have revealed security flaws in Wi-Fi.

  • WEP: Can be cracked in minutes.
  • WPA: Improves WEP with a changing key.
  • WPA2: The latest and most secure version.
(WEP < WPA < WPA2)
