IP (Internet Protocol) address

An IP address can uniquely indentify every computer on a network system. Unlike others, IP address is changeable. An IP address can be 32-bit (IPv4) or 128-bit (IPv6). IPv6 was introduced due to the exhaustion of IPv4.

Uniform Resource Locator (URL)

It is used to identify a specific location on the internet, such as a location of a website, image or file.

Types of Network Protocols

Hypertext Transfer Protocol (HTTP) is a network protocol used by World Wide Web (WWW) to transfer hypertext documents (web pages).

HTTPs is more secure than HTTP because it make use of TLS (Transport Layer Security) encryption. TLS aims at encryption of data during the transmission process in order to secure confidentially, reliably and privacy of communication between clients and servers.

File transfer protocol (FTP) is another common network protocol that its main function is to transfer files over a computer network.

SMTP, POP3 and IMAP are three popular protocols for e-mail communication.

SMTP (Simple Mail Transfer Protocol)
Advantages:
1. Fast email sending
2. Universal standard

Disadvantages:
1. No email retrieval
2. Security vulnerabilities

POP3 (Post Office Protocol 3)
Advantages:
1. Offline email access
2. Simple setup

Disadvantages:
1. No server sync
2. Deletes emails by default

IMAP (Internet Message Access Protocol)
Advantages:
1. Syncs across devices
2. Server email storage

Disadvantages:
1. Requires internet
2. Slower performance
